Project overview

Compare exercises under one controlled setup.

Surface EMG amplitude changes with electrode placement, calibration and signal quality. MyEMG keeps comparisons inside one uninterrupted session.

The solution

Keep every action on the same local reference

Choose a target muscle and at least two actions, complete one relaxed calibration, then record each action without moving the electrodes. Results use the median of valid repetition-level adjusted-envelope means and apply only to that session.

Cheez.sEMG sensor XIAO ESP32C3 BLE wireless Flutter app
How it works

Four steps. One controlled comparison.

01

Choose a target muscle

Select the target muscle and at least two actions to compare.

02

Place and calibrate once

Attach the electrodes, keep the muscle relaxed and establish the session baseline.

03

Record each action

Use the live adjusted-envelope curve and feedback bar while moving at your own rhythm.

04

Review and compare

Confirm reliable rep boundaries, then view this session's actions ordered by their local EMG metric.

Key features

Everything needed for a focused session.

⇄

Within-session Comparison

Compare actions only while electrode placement and calibration remain unchanged.

▥

Live Envelope Feedback

Follow the adjusted-envelope curve and bar without treating either as the final score.

↻

Relaxed Calibration

Establish one local baseline that is reused throughout the comparison.

⌁

Rep Review

Review detected repetitions and request a supported count before confirming an action.

✓

Signal-quality Gates

Reject action windows affected by packet loss, clipping or insufficient quality data.

▦

Session Results

Order valid actions by their repetition-level local EMG metric for this session only.

Device / prototype

Designed to move with you.

The wearable prototype combines a Cheez.sEMG sensor, XIAO ESP32C3 board, battery power and a 3D-printed case designed to be attached to the arm.

  • Wearable target-muscle placement
  • Compact embedded hardware
  • Wireless, untethered feedback

App preview

Live feedback, then a session-scoped result.

The app displays adjusted-envelope feedback during recording, segments repetitions and lets the user review the count. Valid actions are compared only within the current placement and calibration; the result is not a muscle-force percentage or a cross-session claim.

Technical stack

Built across hardware and software.

Hardware

Cheez.sEMG sensor, XIAO ESP32C3, battery-powered wearable prototype and 3D-printed case.

Firmware

Arduino C++, ADC sampling, envelope processing, calibration routine and BLE notify/write.

Flutter, Dart, BLE v2 identification, live envelope feedback, rep review and within-session comparison.

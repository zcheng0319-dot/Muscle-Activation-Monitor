Project overview

Muscle effort, made visible.

MyEMG combines a wearable sensor, embedded processing and a focused mobile experience to turn the biceps' electrical activity into clear, useful feedback.

The system

From muscle signal to live insight

The Cheez.sEMG sensor captures muscle activity while a XIAO ESP32C3 processes the signal and sends it wirelessly to a Flutter app. The result is an intuitive 0–100% activation reading for training, testing and exploration.

Cheez.sEMG sensor XIAO ESP32C3 BLE wireless Flutter app
How it works

Four steps. One live signal.

01

Capture raw EMG

The wearable sensor detects electrical activity produced by the biceps.

02

Process on ESP32C3

The signal is sampled and converted into a calibrated activation percentage.

03

Send via BLE

Live activation and waveform data travel wirelessly to the phone.

04

Visualise in the app

The dashboard presents muscle effort, session metrics and the raw waveform.
